
# Personal Portfolio Website

This is my personal portfolio website built to showcase my skills, projects, and background as a developer.  
The site is lightweight, fast, and fully responsive, designed with simplicity and clarity in mind.

---

##  Features

- Responsive design for desktop and mobile
- Clean and modern UI
- Built with Tailwind CSS
- Easy to maintain and extend
- No heavy frameworks — fast load time

---

## Tech Stack
- HTML5
- Tailwind CSS (v4)
- PostCSS
- Node.js (for tooling only)
